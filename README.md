# 🎮 Угадай число — веб-игра на Flask

Веб-версия классической игры "Угадай число" с тремя режимами, искусственным интеллектом, звуками и таблицей рекордов.

## ✨ Особенности

- 🎯 **Классический режим** — угадай число компьютера
- 🤖 **ИИ угадывает** — загадай число, а компьютер отгадает
- 📊 **Статистика и рекорды** — сохраняются в JSON
- 🔊 **Звуковые эффекты** — через Web Audio API
- 🎚️ **Регулировка громкости**
- 📱 **Адаптивный дизайн** — работает на телефонах

## 🎮 Как играть

1. Выбери режим (Классический / ИИ угадывает)
2. Введи имя и выбери сложность
3. Пытайся угадать число или загадай своё для ИИ
4. Побеждай и попадай в таблицу рекордов!

## 🛠 Технологии

- **Backend:** Python, Flask
- **Frontend:** HTML5, CSS3, JavaScript
- **Звуки:** Web Audio API
- **Хранение данных:** JSON

## 🚀 Быстрый старт

